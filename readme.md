## 🔐 Quick Start

1. Download `install_AhmedJarvis.py`
2. Open CMD and run:
```
python install_AhmedJarvis.py
```
3. Enter password when prompted
4. Jarvis installs and launches automatically

> ⚠️ Requires Python 3.10+ and Git installed on your system.
